ひろたつの独り言

ニュージーランドからつぶやく、ひろたつの独り言コーナーです。

今日の独り言はこちら。（ツイッターから引用）

どどん！！！

実は今滞在しているところはWWOOFではありません。

ただの優しさを頂いています。

仕事をしないでご飯と寝床を提供して頂いています。

ラッキー！とは感じられません。

とても複雑な感覚です。

無償の愛というやつですかね。

今日でここの滞在は最終日となりますが、本当に感謝しかありません。

— ひろたつ@ゼロ円ワーホリでニュージーランド🇳🇿放浪中 (@hirotatsuuu) 2018年12月19日

そうなんです。

今滞在しているところはWWOOFによる滞在ではないのです。

ただの居候なのです。。。

本当に優しい方に巡り会えた。。優しさばかり頂いてて、何も返せていない。。

人生の何処かで頂いた優しさ以上の優しさを返していけるような人になります。

明日使うヒッチハイクボードを作成しました。

今僕はオークランドあたりに滞在しています。

明日向かうのはニュープリマスというところです。

地図を見ると南に330kmほど行ったところです。

順調に行けば明日のお昼ごろには着くかな〜

という感じで
明日からニュージーランド放浪再開です！😋 pic.twitter.com/hk7usfohvQ

— ひろたつ@ゼロ円ワーホリでニュージーランド🇳🇿放浪中 (@hirotatsuuu) 2018年12月19日

英語のお勉強

ここでは、毎日僕が新しく覚えた英語をご紹介します。
僕の英語力の低さが露呈しますが、、しゃーなしｗ

  • Go for it 頑張れ！

今回は一つです。

理由は、この英語が大切だから。
会社を辞めるときに社長から頂いたキーホルダーを今使っている旅のバックパックに付けているのですが、そのキーホルダーに「Go for it」って書いてあります。

頑張れ！

当たって砕けろ！

思いっきりやってみろ！

そんな感じの意味ですね。

僕にとっては大切な言葉の一つです。
だから、今回はこの英語のみにしました。
なぜ今この英語をチョイスしたかって？
明日からニュージーランド放浪を再開するからです！
それに伴って、自分自身を鼓舞するためにも、「Go for it」と自分に言い聞かせます。

ワトルダウンズからおやすみなさい

本日の一日はいかがだったでしょうか。
ワトルダウンズからブログを書くのは今日が最後になります。
明日からはニュープリマスというところに移ります！（ちょっとかっこよくね？ｗ）

てな感じで、
ワーホリや留学を考えてる人、WWOOFやhelpx,workawayを使ってホームステイをしようと考えてる人、お金を使わずに海外に長期滞在しようと考えてる人へ、何かの参考になれば幸いです。

以上、住所不定無職でニュージーランドを放浪しているひろたつ(@hirotatsuuu)の一日でした。

最後まで読んでくださり、ありがとうございました。
僕のニュージーランド放浪はこれからも続きます。
