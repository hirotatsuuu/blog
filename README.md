blog
============================

## Requirements
- hugo v0.54.0

## First
- `hugo`

```
brew install hugo
```


## Devlopment
- `http://localhost:1313`

```
hugo server -D -w
```


## Domain
https://hirotatsu.me


## Usage
- new post

```
hugo new posts/sample.md
```
- create docs

```
hugo
```

- push

```
git push origin master
```
